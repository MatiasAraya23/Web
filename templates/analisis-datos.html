Proyecto Ingenieria de Datos

Librerias Utilizadas: Pandas, Psycopg2, Dotenv.

Data/raw: Directorio que almacena el archivo .csv con aproximadamente 760.000 registros.

Etl/: Modulo encargado del proceso ETL (Extraccion, Tranformacion y Carga) de los datos.

Utils/: Modulo destinado a generar la conexion a la base de datos.

.env: Archivo de configuracion para almacenar credenciales de forma segura y desacoplada del código fuente.

Git: Control de versiones para el seguimiento de cambios en el proyecto. Repositorio
